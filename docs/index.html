The CobBot project aims to revolutionize debt collection by providing an intelligent and automated solution for sending personalized messages to debtors. Its core purpose is to streamline the recovery process, making it more efficient and user-friendly. The system empowers users to create highly customized messages and schedule their automated delivery via WhatsApp, ultimately simplifying debt management and improving recovery rates.

The CobBot's robust backend is built with Python, leveraging the FastAPI API for high performance and scalability. This setup ensures efficient handling of requests and seamless integration of new features. A key aspect of the backend is its intelligent automation: it uses Gemini AI to generate personalized and humanized collection messages, dynamically adapting to each debtor's profile for maximum effectiveness. For persistent data storage, the system utilizes JSON files as its database, offering a lightweight and flexible solution. The backend also handles the automated and scheduled delivery of messages via WhatsApp, employing Python libraries like PyWhatKit to ensure timely dispatch without incurring additional SMS costs. This project was developed as the final assignment for the 2nd edition of the Alura IA Immersion.

The user-friendly interface of the CobBot is crafted with modern web technologies: HTML for structuring the content, CSS for styling, and JavaScript for dynamic interactions. This combination delivers an intuitive and responsive web application. The frontend communicates with the backend via Fetch requests, ensuring a smooth and modern user experience. This design allows users to easily customize messages, schedule deliveries, and efficiently monitor the collection process, all through a clean and accessible interface.
